# 红岩网校移动开发部大一寒假考核（图灵聊天机器人）

## 一：实现功能
#### 1：发送消息和接收消息，将其显示到recyclerview，并隐藏另一个控件，使其两边各一个。
#### 2：接入图灵聊天机器人api，将传入数据并接收返回数据。（感谢红岩网校的学长指点）
#### 3：用Context类中的openFileOutput()方法将聊天数据保存在本地文件里，第二次打开可以查看历史聊天记录。
